    #Pwn Linux (CrunchPwn) is a penetration testing repository/addition for CrunchBang Linux.

    Project maintained by chokepoint Hosted on GitHub Pages — Theme by mattgraham

    Install from repository

    $ curl http://repo.crunchpwn.org/gnupg.key | sudo apt-key add -
$ echo '## Crunchpwn
deb http://repo.crunchpwn.org/ purson main' >> /etc/apt/sources.list
$ sudo apt-get update 
$ sudo apt-get install crunchpwn
